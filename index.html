Welcome to Spectacle¶

Spectacle is a spectroscopic analysis and comparison package built using Python. Its purpose is to provide a well characterized framework for describing and comparing analytic spectra, synthetic spectra (e.g. from hydrodynamic simulations), and real spectra.
